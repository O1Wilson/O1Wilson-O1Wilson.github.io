PolyBurst

PolyBurst is a 2D Block Puzzle game, built in Unity

This game was originally designed in one week for a Game Jam between me and my friends.

The game itself may not be too complicated but it sparked my interest in game development and taught me C#.

I made this a cloned game because I wanted to see if I could replicate something into code just from understanding how it plays.

The entire game is my code, and there are some new features that include: UI Menu system, Unique pieces(5-square), custom randomizer using a tilebag system, and boss battles + rougelike features.
